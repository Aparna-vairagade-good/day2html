Explain br tags?

  • br: The Line Break element. The br HTML element produces a line break in text (carriage-return). It is useful for writing a poem or an address, where the division of lines is significant.

  • In this article, we will know how to add line-break using HTML br tag. The browser does not recognize new line and paragraph formatting in the text. If you want to start a new line, you need to insert a line break with the help of the br. The br tag inserts a single carriage return or breaks in the document. This element has no end tag.
